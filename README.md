# Парсер новостей

Проект для парсинга новостных сайтов и группировки новостей по тематическим разделам.

## О проекте

Программа извлекает новости с указанного URL, определяет их тематику и группирует по разделам (Спорт, Политика, Экономика и т.д.). Каждая новость содержит заголовок и ссылку на полный текст.

## Зависимости

Для работы проекта необходимы следующие библиотеки:

- beautifulsoup4
- requests

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите

### Парсинг страницы по умолчанию (архив Iz.ru)
```bash
python parse_news.py
```

### Парсинг произвольной страницы
```bash
python parse_news.py "https://example.com/news"
```

### Сохранение в указанный файл
```bash
python parse_news.py -o my_news.json "https://example.com/news"
```
